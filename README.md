## Keep Me Up – Football PWA

The app that keeps you up to date with your favourite team, anytime, anywhere.

Keep Me Up is a Progressive Web Application built using Ionic and Angular that provides real-time football data including league tables, team details, and match information.

The application allows users to:
- View league tables with detailed statistics and recent form
- Explore team information including badges and performance data
- Navigate between pages using Angular routing
- Save preferences such as favourite team, league, and theme using local storage
- Switch between light, dark, and team-based themes

The app consumes external football data via a public API and presents it in a clean, responsive UI designed for both desktop and mobile use.

This project demonstrates:
- Angular standalone components
- Data binding (interpolation and two-way binding)
- API integration using Observables and HttpClient
- Persistent storage using Ionic Storage
- Modular service-based architecture
